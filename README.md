### 简历 (brilliant-cv)

基于 [brilliant-cv](https://github.com/yunanwg/brilliant-CV) 的 Typst 简历模板，支持中/英/日三语。

```bash
cd brilliant-cv

# 中文简历
typst compile cv.typ output/cv-zh.pdf --input profile=zh

# 英文简历
typst compile cv.typ output/cv-en.pdf --input profile=en

# 日语简历
typst compile cv.typ output/cv-ja.pdf --input profile=ja

# 实时预览
typst watch cv.typ output/cv-zh.pdf --input profile=zh
```

修改资料 → 编辑 `profile_zh/`（中文）、`profile_en/`（英文）、`profile_ja/`（日语）。
